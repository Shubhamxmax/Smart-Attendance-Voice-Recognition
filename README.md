#  Smart Attendance Voice Recognition

This project is a **smart attendance system** that uses **human voice** to mark attendance automatically.  
Instead of calling names or signing sheets, the system recognizes a person by their voice.

---

## About the Project

The goal of this project is to:
- Identify a person using their voice
- Mark attendance automatically
- Reduce fake or proxy attendance

Voice is used as a **biometric**, just like fingerprints or face recognition.

---

### Dataset Details

- Each speaker has many audio files
- Audio format is **WAV**
- Voices are collected from public sources like YouTube and audiobooks
